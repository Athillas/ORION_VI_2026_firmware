# EN
## !! IMPORTANT !!
Before starting work on the project, please carefully review the documentation, especially the files `docs/design_doc_pl.md` (in Polish) or `docs/design_doc_en.md` (in English). These files contain explanations regarding the module architecture to be used in this project (docs are currently WIP).

## Description
This directory contains Arduino firmware code in C++, written for four ESP32 WROOM boards: `Vacuum`, `Power`, `RoboticArm`, and `RockerInterface`, schematics for each board (see `schematics`), and documentation (see `docs`, currently WIP).

## Opening the Project

To open individual projects, use the [PlatformIO extension for VSCode](https://platformio.org/). Each project directory contains a `platformio.ini` file that defines the platform, architecture and libraries, used in the module.

Do otwierania poszczególnych projektów należy używać rozszerzenia [PlatformIO dla VSCode](https://platformio.org/). Każdy katalog projektu zawiera plik `platformio.ini`, który określa platformę, architekturę oraz biblioteki używane w danym module.
